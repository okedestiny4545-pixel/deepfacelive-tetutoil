# Windows 10 Quick Setup Tutorial

## Prerequisites

### 1. Install Latest Graphics Drivers for Your Video Card

Ensure your GPU drivers are up to date for optimal performance.

![Video Card Icon](video_card_icon.png)

---

### 2. Set Total Virtual Memory to 32GB or More

Configure your system's virtual memory settings to at least 32GB.

![Virtual Memory Settings](virtual_memory.jpg)

---

## Installation Steps

### 3. Download Windows Build EXE

Download the latest Windows build executable from one of the available sources.

![Download Build](download_build.jpg)

---

### 4. Run and Unpack to Root of Any Disk

Extract the downloaded file to the root directory of any disk on your system.

![Run and Unpack](run_and_unpack.jpg)

---

### 5. Run the DeepFaceLive.bat File

Execute the `_DeepFaceLive.bat_` file to launch the application.

![Run DFL Batch File](run_dfl_bat.jpg)

---

## Configuration

### 6. Select Video Source

Choose from one of the following options:

#### Option A: Video File from Samples

Select a **Video file** from the available samples in **File source**.

![Select Video File](select_video_file.jpg)

#### Option B: Camera Source

Alternatively, select **Device index** and **Driver** in **Camera source**.

![Select Camera Source](select_camera_source.jpg)

---

### 7. Choose Face Detector

Select your preferred **Face detector** and assign a **Device** for processing.

![Choose Face Detector](choose_face_detector.jpg)

---

### 8. Choose Face Marker

Select your preferred **Face marker** and assign a **Device**.

![Choose Face Marker](choose_face_marker.jpg)

---

### 9. Choose Face Swapper

Select the **Model** and **Device** in the **Face swapper** module.

![Choose Face Swapper](choose_face_swapper.jpg)

---

## Output Configuration

### 10. Preview the Swapped Face

You should now see the swapped face in the **Final frame** preview window.

![Final Frame Preview](final_frame.jpg)

---

### 11. Configure Stream Output

In **Stream output**, choose:
- **Mode**: `Merged frame`
- Press the **window** button to display the output.

![Stream Output Settings](stream_output.jpg)

---

### 12. View DeepFaceLive Output

The **DeepFaceLive output** window will now display the processed video stream.

![DeepFaceLive Output Window](dflive_output.jpg)

---

## Fine-Tuning

### 13. Optimize Performance and Quality

Experiment with various parameters in the different modules to achieve the best performance and quality for your use case.

![DeepFaceLive Main Window](deepfacelive_window.jpg)

---

## Tips for Best Results

- Ensure adequate GPU VRAM for smooth processing
- Test with different face detector and marker models
- Adjust device assignments based on your hardware capabilities
- Monitor resource usage while adjusting parameters
- Start with default settings and gradually fine-tune

---

**Happy face swapping!** 🎭
